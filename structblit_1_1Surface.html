◆ load_read_only()

Surface * blit::Surface::load_read_only ( const packed_image *  image)
static

Similar to load, but the resulting Surface points directly at the image data instead of copying it. data should not be modified after loading, so no drawing can be done to this surface. If the image is paletted, the palette can still be modified.

Only works for non-packed images.

Parameters
image
Returns
Surface containing loaded data or nullptr if the image was invalid
